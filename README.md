Requirements:
	1. Install Composer and Laravel
	2. Database Set up:-
		DB_CONNECTION=mysql
		DB_HOST=localhost
		DB_PORT=3306
		DB_DATABASE=nri
		DB_USERNAME=lovepreet
		DB_PASSWORD=Singh
	
	3.Database Migrate:
		a. Run PHP and Mysql Server
		b. Open cmd/terminal as administrator
		c. Navigate to project directory
		d. run these commands in cmd/terminal 
			->php artisan migrate
			->php artisan serve
			
	4. Open the link in your browser which you will get as an output of second command.
